<h1>Agro Advance</h1>

<p>Agro Advance is a simple agriculture-support tool that gives basic crop suggestions based on climate and helps users understand what grows best in different weather conditions. The goal is to make farming decisions easier using clean data and a user-friendly interface.</p>

<h4>Features</h4>

Weather-based crop recommendations

Simple and easy-to-use UI

Can be expanded with more features in the future

<h4>Tech Used</h4>

HTML, CSS, JavaScript, Python (if used), Weather/API data
